# Tic-Tac-Toe

A digital implementation of the classic Tic-Tac-Toe game developed in Java using Maven. This project is part of a software engineering exercise focusing on the complete software development lifecycle, including planning, implementation, testing, version control, and CI/CD.

## Project Overview

As a new game studio, our goal is to design, develop, test, and deploy a digital version of Tic-Tac-Toe while following professional software engineering practices.

## User Stories

### US-001: Make a Move

As a player, I want to be able to make a move by choosing an empty square, so that I can place my symbol on the board.

### US-002: View Current Game State

As a player, I want to be able to see the current state of the game, so that I can keep track of the moves made by both myself and my opponent.

### US-003: Game Result Notification

As a player, I want to be notified when the game has ended in a win, loss, or draw, so that I can see the result of the game.

### US-004: Start a New Game

As a player, I want to be able to start a new game after the current game has ended, so that I can play again.
